    Operation           |       Windows Command	      |     Linux Command
________________________|_____________________________|____________________________
List files in directory |   dir                       |   ls
Navigate to directory   |   cd path\to\your\project	  |   cd /path/to/your/project
Create virtual env      |	python -m venv myenv	  |   python3 -m venv myenv
Activate virtual env    |	myenv\Scripts\activate	  |   source myenv/bin/activate
Deactivate virtual env  |	deactivate	              |   deactivate
